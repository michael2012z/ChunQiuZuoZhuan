# 僖公二十年

## 經 <a name="05Xi20Jing"></a>

<a name="05Xi20Jing01">[僖20經01]</a> 二十年春，新作南門。

<a name="05Xi20Jing02">[僖20經02]</a> 夏，郜子來朝。

<a name="05Xi20Jing03">[僖20經03]</a> 五月，乙巳，西宮災。鄭人入滑。

<a name="05Xi20Jing04">[僖20經04]</a> 秋，齊人、狄人盟于邢。

<a name="05Xi20Jing05">[僖20經05]</a> 冬，楚人伐隨。

## 傳 <a name="05Xi20Zhuan"></a>

<a name="05Xi20Zhuan01">[僖20傳01]</a> 二十年春，新作南門。書，不時也。凡啟塞從時。

<a name="05Xi20Zhuan02">[僖20傳02]</a> 滑人叛鄭，而服於衛。

<a name="05Xi20Zhuan03">[僖20傳03]</a> 夏，鄭公子士、洩堵寇，帥師入滑。

<a name="05Xi20Zhuan04">[僖20傳04]</a> 秋，齊狄盟于邢，為邢謀衛難也。於是衛方病邢。

<a name="05Xi20Zhuan05">[僖20傳05]</a> 隨以漢東諸侯叛楚。

<a name="05Xi20Zhuan06">[僖20傳06]</a> 冬，楚鬭穀於菟帥師伐隨，取成而還。

<a name="05Xi20Zhuan07">[僖20傳07]</a> 君子曰：「隨之見伐，不量力也。量力而動，其過鮮矣。善敗由己，而由人乎哉？《詩》曰：『豈不夙夜，謂行多露。』」

<a name="05Xi20Zhuan08">[僖20傳08]</a> 宋襄公欲合諸侯。臧文仲聞之曰：「以欲從人則可，以人從欲鮮濟。」
